Objectif

Dans ce jeu, vous contrôlez un robot dans un labyrinthe. L'objectif est de collecter des bombes et de les utiliser pour peindre le sol.
Le labyrinthe se compose de 6 lignes et 10 colonnes. Vous devez guider votre robot pour qu’il récupère des bombes, les dépose dans le labyrinthe, puis fuie pour éviter l’explosion. Lorsque une bombe explose, elle colorie le sol sous elle et 1 cases de chaque côté en forme de croix dans chaque direction non bloqué par un mur.
Chaque bombe collecté rapporte 1 points et chaque case coloré rapporte aussi 1 points.

Pour l'emporter, il faut atteindre le nombre de points requis, écrit en haut à gauche de l'écran et sans rentrer dans un mur ou se faire exploser sois même.
Conditions de Victoire
  • Vous atteignez le nombre de points requis.
Conditions de Défaite
  • Le robot se crash dans un mur.
  • Le robot est pris dans l'explosion d'une bombe.
  • Vous n'atteignez pas les points en 200 turns.
  • Vous ne répondez pas dans les temps ou la commande est inconnue.
Entrée d'initialisation
- Ligne 1: Un entier : rows, représentant le nombre de lignes du labyrinthe.
- Ligne 2: Un entier : columns, représentant le nombre de colonnes du labyrinthe.
- Ensuite, une ligne par cellule du labyrinthe, contenant chacune 5 booléen séparé par un espace : west, east, north, south, et hasBomb.

Ces valeurs indiquent la présence de murs dans chaque direction et si une bombe est présente dans la cellule.
Entrée pour un tour de jeu
- Ligne 1: 4 entiers séparé par un espace : robotX, robotY, robotA et robotBombs.

Ces valeurs représentent la position actuelle du robot, son orientation et le nombre de bombes qu’il transporte.

L’orientation robotA est définie comme suit :
  • 0 → regarde vers l’OUEST
  • 1 → regarde vers l’EST
  • 2 → regarde vers le NORD
  • 3 → regarde vers le SUD
Sortie
Affichez l'une des actions suivantes :
  • MOVE_FORWARD : Fait avancer le robot d’une cellule. S’il rencontre un mur, il s’écrasera.
  • MOVE_BACKWARD : Fait reculer le robot d’une cellule. S’il rencontre un mur, il s’écrasera.
  • TURN_LEFT : Fait pivoter le robot de 90° vers la gauche (sens antihoraire).
  • TURN_RIGHT : Fait pivoter le robot de 90° vers la droite (sens horaire).
  • DROP : Dépose une bombe dans la cellule actuelle, si le robot en possède au moins une et qu’aucune bombe n’est déjà présente dans cette cellule.
  • WAIT : Attend le prochain tour.
Contraintes
  • 0 ≤ robotX < 10
  • 0 ≤ robotY < 6
  • 0 ≤ robotA ≤ 3
  • 0 ≤ robotBombs
