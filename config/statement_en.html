Goal

In this game, you control a robot in a maze. The goal is to collect bombs and use them to paint the ground.
The maze consists of 6 rows and 10 columns. You must guide your robot to collect bombs, drop them in the maze, and run to avoid the explosion.
To earn points, collect as many bombs as possible and paint the largest possible area of the maze.
Initialization Input
- Two integers: rows and columns, representing the number of rows and columns in the maze.
- Then, one line per cell in the maze, each containing 5 values: west, east, north, south, and hasBomb. These represent the presence of walls (1 or 0) in each direction and whether a bomb is present in that cell (1 or 0).
Input for a Game Round
- Four integers: robotX, robotY, robotA, and robotBombs.
These represent the robot's current position, orientation, and the number of bombs it is carrying.

The orientation robotA is defined as:
  • 0 → facing WEST
  • 1 → facing EAST
  • 2 → facing NORTH
  • 3 → facing SOUTH
Output
Output one of the following actions:
  • MOVE_FORWARD: Moves the robot one cell forward. If it moves into a wall, it will crash.
  • MOVE_BACKWARD: Moves the robot one cell backward. If it moves into a wall, it will crash.
  • TURN_LEFT: Rotates the robot 90° to the left (counterclockwise).
  • TURN_RIGHT: Rotates the robot 90° to the right (clockwise).
  • DROP: Drops a bomb in the current cell, if the robot has at least one bomb and there is no bomb already present in that cell.
Constraints
  • rows = 6
  • columns = 10
  • 0 ≤ robotX < 10
  • 0 ≤ robotY < 6
  • 0 ≤ robotA ≤ 3
